# BSGP 7030 assignment 2

Jupyter notebooks and executable scripts to generate a simple linear regression of salary data based on years of experience.

## Usage

Jupyter notebooks are included to allow for interactive editing. Executable scripts (.py and .R) can be run in the command line using the following syntax:

```bash
python linear_regression_python.py <data.csv> <x column name> <y column name>
Rscript linear_regression_r.r <data.csv> <x column name> <y column name>
```

Executable scripts output model fit to the terminal and export a graph of the data and regression model.
